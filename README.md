# 🧠 AI-Powered Learning Assistant

A full-stack web application that transforms standard PDF documents into interactive learning experiences using Google's Gemini AI. 

Users can upload their study materials and instantly generate AI-powered flashcards, comprehensive quizzes, detailed document summaries, and even chat directly with their document to explain complex concepts.

**Live Demo:** [https://ai-based-learning-app-644o.vercel.app/dashboard](https://ai-based-learning-app-644o.vercel.app/dashboard)
**Backend API:** `https://ai-based-learning-app-1.onrender.com`

---

## ✨ Key Features

* **📄 Smart Document Parsing:** Upload PDFs and instantly extract readable text.
* **📇 AI Flashcard Generation:** Automatically generate study flashcards based on the specific context of your document.
* **📝 Automated Quizzes:** Test your knowledge with dynamically generated quizzes.
* **💬 Chat with your Document:** Ask questions and get explanations directly sourced from your uploaded materials.
* **🔒 Secure User Authentication:** Full login and registration system using JWT.

---

## 🛠️ Tech Stack

**Frontend:**
* React.js (Built with Vite for speed)
* Tailwind CSS (Styling and responsive design)

**Backend:**
* Node.js & Express.js (RESTful API architecture)
* MongoDB & Mongoose (Database and object modeling)
* Google Gemini API (Powering the AI generation features)
* Multer & PDF-Parse (File uploading and text extraction)
